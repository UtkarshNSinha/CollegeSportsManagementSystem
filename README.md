College Sports Management System

Tech Stack:
- Java
- Oracle SQL
- PL/SQL
- JDBC

Description:
A database-driven application to manage college sports events,
participants, teams, schedules, and results.

Key Features:
- Normalized relational schema with constraints and triggers
- PL/SQL procedures for registration and scoring
- Java application using JDBC for database interaction

How to Run:
- Import SQL scripts into Oracle
- Configure DB connection in Java
- Run main class
